# /btw 旁路问答

`/btw`（by the way）是一个专门应对主会话被长任务占用时的旁路快问能力。发送 `/btw <问题>` 可以绕过当前会话锁，立即获得一个独立的工具无关快答。

## 使用方式

在私聊或群聊中，直接发送：

```
/btw 今天上海天气怎么样
```

插件会识别 `/btw` 前缀并走专门的旁路投递链路。

## 适用场景

- 主会话被超长回复或复杂工具链占用时，想快速插一个问题
- 需要一个与当前会话上下文隔离的临时问答（旁路问题不会污染会话历史）

## 不适用场景

- 需要依赖工具调用（function call）的任务——旁路查询不携带 tool 权限
- 需要承接上文语境的补充提问——旁路回复独立投递，不会进入主会话

## 行为细节

- 旁路查询绕过 session lock，不会被主会话排队阻塞
- 旁路回复不走 AI Card 链路，始终以 markdown 文本独立投递
- 旁路查询不保留长期内存，只做一次性回答

## 配置

无需额外配置，安装插件后即可使用。
