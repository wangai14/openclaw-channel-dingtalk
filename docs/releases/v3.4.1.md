# v3.4.1 发布说明 🚀

> [!IMPORTANT]
> **核心兼容性变更提醒 (Breaking Change)**
> 本次版本主要用于跟进 OpenClaw `v3.22` 的上游 Breaking Change。
> **从 `v3.4.1` 开始，本插件要求的最低 OpenClaw 兼容版本为 `v3.22`。**
> 如果你暂时不打算升级 OpenClaw 主程序，请继续停留在 `v3.4.0` 及以前版本。

本次 `v3.4.1` 是一个关键的兼容性与稳定性补丁版本。除了完成对上游架构的适配外，还修复了状态流转和沙箱环境下的媒体发送问题。

## 🚀 架构重构与适配

* **同步上游 Plugin SDK 新版 API**
  * `PR #395`（by @soimy）全面跟进 OpenClaw `v3.22` 的底层接口变更，完成核心通信与插件机制的重构适配。

## 🛠 修复与稳定性改进

* **修复 `FAILED` 终态下的等待阻塞问题**
  * `PR #399`（by @BrilliantWang）解决了状态机进入 `FAILED` 后 `waitForStop` 无法正常 resolve 的问题。
* **修复沙箱环境下工作区文件发送失败**
  * `PR #398`（by @zhumin-zizhu，Fixes `#397`）修复了 Sandbox 模式下 `sendMedia` 无法正确处理 workspace 媒体文件的问题。
