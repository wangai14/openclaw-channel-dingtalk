# v3.6.5 发布说明

本次 patch 版本新增 **DingTalk ask-user 问答卡片**，并补齐相关回调、超时与测试覆盖。

**最新版本入口**：[`latest.md`](./latest.md)

## ✨ 新增功能

* **新增 DingTalk ask-user 问答卡片**
  * [`PR #583`](https://github.com/soimy/openclaw-channel-dingtalk/pull/583)（by @BenGuanRan）新增 DingTalk 侧的 ask-user card 工具注册、卡片模板与回调处理，让运行时可以在 DingTalk 内向用户发起表单式追问。
  * 支持表单字段 schema、用户提交、取消操作、超时通知与合成消息回注，便于 agent 在需要补充信息时继续当前会话。

## 🛠 修复与稳定性

* **强化 ask-user 卡片生命周期**
  * 增加超时 tombstone、空表单提交、取消标记解析与回调状态对齐处理，降低重复回调或过期回调导致的卡住风险。
  * 补齐 `ask-user-question`、卡片 action handler 与 inbound card 流程的单元测试覆盖。
